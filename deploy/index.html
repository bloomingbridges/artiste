l'artiste template

Gir loves tacos (2012)

Animation on canvas

l'artiste is a minimal, (almost) production-ready template for HTML5 canvas development that comes with an aspect-ratio agnostic, vertically-ruled, centric layout and Grant Skinner's set of CreateJS libraries out of the box (via CDN).
Includes EaselJS 0.4.2, TweenJS 0.2, SoundJS 0.2 and PxLoader 0.1.

Special thanks go to Grant Skinner (@gskinner), Justin Windle (@soulwire), Alexis Sellier (@cloudhead) and Robert Bringhurst.
